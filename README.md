### 🏗️ Featured Projects

**[Torrimac POS](https://github.com/Nimuthu) — Offline-First POS & Pharmacy Inventory System**
`Electron` `React 19` `TypeScript` `SQLite` `Prisma` `Supabase (PostgreSQL)`
Full-stack desktop POS built from the ground up for offline-first operation with automatic hourly bidirectional cloud sync. Features a secure IPC layer with Zod validation and role-guarded handlers (Admin/Supervisor/Cashier), pharmacy compliance (medicine expiry enforcement, batch tracking), bcrypt PIN auth with rate limiting, cash shift reconciliation (Z-Reports), and USB barcode scanner integration.

**GPOS Admin — Admin Dashboard for POS System**
`Next.js` `TypeScript` `Supabase (PostgreSQL)` `Zod` `iron-session`
Web-based admin panel for staff, inventory, and sales management for the Torrimac POS ecosystem. Includes cookie-based session auth, CRUD interfaces for employees and products, transaction reporting, and a mid-project security migration moving credential hashing from client-side SHA-256 to server-side bcrypt via Postgres `SECURITY DEFINER` functions — plus a GitHub Actions CI/CD pipeline with Trivy scanning.

**[RydeFlexi](https://rydeflexi.com) — Car Rental & Ride-Hailing Platform**
`Front-End Development`
Contributed to a car rental and ride-hailing ecosystem automating the vehicle leasing lifecycle — driver onboarding workflows, automated rental requests, a smart pricing engine, and PayHere payment integration.
